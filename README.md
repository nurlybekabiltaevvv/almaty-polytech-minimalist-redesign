## Overview

**Almaty Polytech — Minimalist Redesign** is a modern conceptual redesign of the Almaty Polytechnic College website.

The project focuses on clean visual hierarchy, minimalist layout, strong typography, and modern UI/UX principles.  
It is built as a fast and optimized Single Page Application using the latest frontend technologies.

🔗 **Live Demo:**  
https://nurlybekabiltaevvv.github.io/almaty-polytech-minimalist-redesign/


## Objectives

- Deliver a modern and minimal visual identity
- Improve content readability and layout structure
- Emphasize academic programs and statistics
- Ensure responsive and adaptive design
- Apply scalable component-based architecture


## Tech Stack
React 19 — UI development
Vite 7 — Build tool & development server
Tailwind CSS 4 — Utility-first styling
TypeScript — Type safety
Lucide Icons — Icon system
GitHub Actions — CI/CD deployment
GitHub Pages — Hosting


## Architecture

- Clean separation of UI components
- Utility-first styling via Tailwind
- Optimized Vite production build
- Automated deployment pipeline

### Clone repository

git clone https://github.com/nurlybekabiltaevvv/almaty-polytech-minimalist-redesign.git
cd almaty-polytech-minimalist-redesign

### Install dependencies
npm install

### Start development server
npm run dev

### Build for production
npm run build

### Preview production build
npm run preview


## Deployment

The project uses **GitHub Actions CI/CD pipeline**.

On every push to the `main` branch:
1. Dependencies are installed
2. The project is built via Vite
3. The `dist` folder is deployed automatically to GitHub Pages

Workflow file:


.github/workflows/deploy.yml
